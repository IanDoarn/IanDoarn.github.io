Pygorithm

Data structures and Algorithms written entirely in Python

Pygorithm is an education package for beginner programmers to help introduce them to the concepts of data structures and algorithms. All of the most important subject are covered and more are added all the time!

Requirements:

Python>=3.0 |

ZBSMSA

ZimmerBiomet Surgery Management System Automation

zbsmsa is automation tool for interacting with ZimmerBiomets SMS inventory site: https://www.zimmersms.com/ Using Selenium, BeautifulSoup, and the chrome driver, zbsmsa can execute commands, send keys, and can be used to create complex macros to automate tasks.

Requirements:

psycopg2>=2.7.1 | selenium>=3.3.1 | cssselect>=1.0.1 | lxml>=3.7.3 |
  • ZBSMSA on GitHub
  • Written in: Python

Ryport

Postgres to Excel/CSV

Ryport is a softwrapper for Postgres built on top of the Python package psycopg2. It is an extremely simple package that make is easy for users to execute queries and quickly write information to portable / readable formats such as Excel and CSV files.

Ryport may support other formats in the future, while I did plan on also implimenting MySQL and Oracle, I never quite go that far!

Requirements:

psycopg2>=2.7.1 | mysql-connector-python>=2.0.4 | xlsxwriter>=0.9.6 | lxml>=3.7.3 |
  • Ryport on GitHub
  • Written in: Python
